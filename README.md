# PortrAid - AI Portrait Composition Assistant

A real-time portrait composition analyzer that helps users take better portraits by providing instant feedback on composition elements like rule of thirds, eye line, headroom, and more.

## Features
- Real-time face detection and analysis
- Professional composition guidelines
- Live feedback with visual guides
- Multiple composition metrics
